Sekarang saya ingin anda melakukan perubahan terkait proses backup data menjadi doubel database. Database pertama tetap menggunakan TiDB dan database kedua menggunakan appscript. Sehingga pada awalnya logika backup data adalah sebagai berikut app script mengambil data dari TiDB. Sekarang saya ingin front end melakukan pengiriman data langsung ke spreadsheet. Maka sekarang saya ingin anda memperbarui autosave-c.js dan autosave-f.js untuk mendukung ide saya ini. Dan karena sekarang frontend akan bekerja ektra karena harus mengirimkan 2 data ke 2 sumber yang berbeda yaitu TiDB dan google spreadsheet. Maka saya ingin anda memikirkan cara agar integrasi kedua hal ini dapat berjalan dengan cepat dan efektif tanpa hambatan sedikitpun. Oleh karena itu karena ini adalah konsep baru dan untuk menghindari error kita akan melakukan uji coba khusus Line 3 terlebih dahulu untuk melihat apakah data yang dikirimkan oleh front end ke TiDb dan google spreadsheet dapat berjalan dengan cepat dan efisien.
